== Description ==

The <strong>WPTrainMe User Tutorials plugin</strong> provides instant access to hundreds of detailed WordPress step-by-step tutorials from your WordPress Dashboard. 

WPTrainMe is a comprehensive WordPress user training system. Our WordPress tutorials have been specifically written for users with no coding skills who want to learn WordPress. All tutorials are organized in logical sequence and cover the following areas of WordPress:

<ol>
	<li style="text-align: justify;"><strong>WordPress Introduction</strong>: An introduction to WordPress for new users.</li>
	<li style="text-align: justify;"><strong>WordPress Installation</strong>: Learn how to install WordPress. Includes pre-installation and post-installation checklists.</li>
	<li style="text-align: justify;"><strong>WordPress Configuration</strong>: Learn how to set up a WordPress site and configure WordPress settings.</li>
	<li style="text-align: justify;"><strong>How To Use WordPress</strong>: Learn how to use all of the main features of WordPress. Includes tutorials on using the WordPress App. for mobile devices.</li>
	<li style="text-align: justify;"><strong>WordPress Plugin Tutorials</strong>: Learn how to install, configure and use dozens of essential WordPress plugins.</li>
	<li style="text-align: justify;"><strong>WordPress Customization</strong>: Learn how to customize various aspects of your WordPress site.</li>
	<li style="text-align: justify;"><strong>WordPress Management</strong>: Learn how to manage and keep your WordPress site fully maintained and up-to-date.</li>
	<li style="text-align: justify;"><strong>WordPress Optimization</strong>: Learn how to keep your WordPress site fully optimized and increase site performance.</li>
	<li style="text-align: justify;"><strong>WordPress Security</strong>: Learn how to keep your WordPress site secure and protected.</li>
	<li style="text-align: justify;"><strong>WordPress Strategies</strong>: Learn how to grow and profit with WordPress. Includes strategies for driving more traffic, increasing conversions, integrating with social media, monetizing WordPress, selling products online, creating membership sites, etc.</li>
</ol>

The <strong>FREE WPTrainMe plugin edition</strong> provides access to over 70 essential WordPress How-To tutorials. It is ideal for helping new users learn how to use WordPress and can also be used by WordPress consultants and service providers to train new clients.

Our complete library of WordPress tutorials can be accessed from your dashboard with the WPTrainMe plugin by upgrading to our <strong>PRO Edition</strong>. PRO gives you access to all written tutorials, plus hundreds of video tutorials and downloadable files, guides, flowcharts, etc. For more details, visit <strong><a title="WPTrainMe.com PRO Plugin" href="http://wptrainme.com/wordpress-tutorials-plugin-pro" target="_blank">WPTrainMe.com PRO</a></strong>. 

All written WordPress tutorials can be viewed online by visiting <strong><a title="WPCompendium.org - Free WordPress Tutorials For Beginners" href="https://wpcompendium.org" target="_blank">WPCompendium.org</a></strong>.

For more details about WPTrainMe, visit <strong><a title="WPTrainMe.com - WordPress Training Plugin" href="http://wptrainme.com" target="_blank">WPTrainMe.com</a></strong>.

== Frequently Asked Questions ==

<strong>Are The WordPress Tutorials In The WPTrainMe Plugin Kept Up-To-Date?</strong>

As any experienced WordPress user will appreciate, WordPress is constantly changing and evolving. We provide hundreds of detailed written tutorials with thousands of screenshots, so keeping everything up-to-date can be very challenging. We update our tutorials when WordPress introduces changes that affect the functionality of core features. 

<strong>Where Are The Tutorials Stored?</strong>

The WPTrainMe WordPress training system comprises of hundreds of comprehensive tutorials with detailed written information, thousands of screenshot images, hundreds of videos, plus downloadable templates, checklists, flowcharts, additional guides, etc. To effectively manage and keep all of this information updated, we store all content on our servers and make the tutorials accessible to WordPress users via the plugin. 

Essentially, the WPTrainMe plugin acts as an interface to the training content that consumes almost no resources and provides users with a convenient way to access the tutorials from their WordPress dashboard. 

<strong>Why Do I Need An Activation/Authentication Key?</strong>

Our server needs to know which plugin edition you have downloaded (i.e. Free or PRO), so it can deliver you the correct tutorials for your plugin edition. 

FREE Edition plugin users can get an activation key simply by registering for a FREE WPTrainMe membership here: <strong><a title="WPTrainMe.com - WPTrainMe WordPress Training Plugin" href="http://wptrain.me/register/free" target="_blank">WPTrainMe Plugin Registration</a></strong> and using their login details to authenticate the plugin and activate access to the tutorials.

Authentication instructions for PRO users are available in our <strong><a title="WPTrainMe.com - Plugin User Guide" href="http://wptrainme.com/docs" target="_blank">Plugin User Guide</a></strong>.

<strong>Can I Use The FREE Plugin To Train My Clients?</strong>

Absolutely! The FREE edition of the WPTrainMe plugin provides dozens of WordPress tutorials designed to help new users learn all the basics of using WordPress. This lets you focus on what you do best while providing your clients with effective training on how to use their WordPress sites.

<strong>Can I View The WordPress Tutorials Without Downloading The Plugin?</strong>

Yes. To view all of our detailed written WordPress tutorials online, visit <strong><a title="WPCompendium.org - Free WordPress Tutorials For Beginners" href="https://wpcompendium.org" target="_blank">WPCompendium.org</a></strong>. The WPTrainMe plugin provides users with a convenient way to access our written tutorials, plus video tutorials and downloadable templates, checklists, flowcharts, etc. from their dashboard and get help while working on their sites.
